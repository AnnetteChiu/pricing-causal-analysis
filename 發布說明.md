## 🎯 立即使用

### 克隆項目
```bash
git clone https://github.com/AnnetteChiu/pricing-causal-analysis.git
cd pricing-causal-analysis
pip install -r requirements.txt
```

### 快速演示
```bash
# 模擬數據演示
python demo_analysis.py

# 真實數據演示
python real_data_analysis.py
```

### 使用你的數據
```python
from real_data_analysis import analyze_real_data

column_mapping = {
    '你的價格列名': 'price',
    '你的銷量列名': 'sales_volume'
}

analyzer, data = analyze_real_data('your_data.csv', column_mapping)
```

## 📊 項目亮點

### 🆕 真實數據支持
- **智能數據加載** - 自動處理編碼和格式問題
- **列名自動識別** - 智能建議列名映射
- **完整預處理** - 缺失值、異常值、數據驗證
- **自動變數創建** - 智能生成分析所需變數

### 🔬 專業分析方法
- **隨機實驗** (A/B測試) - 金標準因果識別
- **回歸調整** - 控制混淆變數
- **傾向得分匹配** - 處理選擇偏誤
- **差分差分法** - 時間序列因果分析
- **工具變數法** - 處理內生性問題
- **回歸不連續** - 閾值規則分析
- **機器學習方法** - Double ML和異質性分析

### 📈 實用價值
- **科學決策** - 基於數據而非直覺
- **風險控制** - 避免錯誤定價
